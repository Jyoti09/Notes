  • For my menu bar it is better to use justify-content just for center alignment, flex-start or flex end.
  • To have space between and around the flex items it is better to use margins. I believe, atleast for this project.
  • Use justify-content with flex-direction:row
  • Use align-content with flex-direction:column
  • However, align-items is used to align flex-items vertically.
  • For all above flex-container property, we can use center, flex-start, flex-end,space-around, space-between as the values
  • stretch is for align-items only.
